🏥 Hospital Emergency Room Dashboard – Power BI

This project presents a comprehensive Hospital Emergency Room Analytics Dashboard built using Power BI to monitor patient flow, operational efficiency, and service performance. The dashboard enables healthcare stakeholders to track key metrics such as patient wait time, admission rates, satisfaction scores, and departmental workload.

📌 Project Overview

The dashboard is designed to provide both high-level insights and detailed patient-level data across multiple views. It helps hospital management improve decision-making, reduce wait times, and optimize emergency room operations.

📈 Key Features
Interactive multi-page dashboard with navigation panel
Dynamic filtering by year, month, and date range
KPI cards for quick performance monitoring
Drill-through functionality for detailed patient analysis
Heatmaps and charts for identifying peak hours and delays
Clean and structured UI for healthcare reporting
🛠️ Tools & Technologies
Power BI Desktop – Data visualization
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – KPI calculations
Data Model:
Fact Table: Patient Records
Dimension Tables: Date, Department, Demographics
🎯 Project Objectives
Monitor emergency room performance in real-time
Reduce patient wait times and improve service efficiency
Track admission rates and referral patterns
Analyze patient demographics for better resource planning
Ensure service-level compliance (patients seen within 30 minutes)
📊 Key Insights
Approximately 50% of patients are admitted, indicating balanced ER intake
Average wait time ranges between 35–37 minutes, highlighting optimization opportunities
A significant portion of patients are not seen within 30 minutes, impacting service KPIs
Certain departments (e.g., General Practice) receive the highest referrals
Peak patient loads vary by day and hour, critical for staff scheduling
